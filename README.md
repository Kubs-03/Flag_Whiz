<h1>Flag_Whiz</h1>
<p>Test your knowledge with the fun and challenging flag guessing game! Identify flags from countries all around the world and improve your geography skills. 
  Perfect for trivia lovers and geography enthusiasts.
  The game includes interesting features such as <strong> random question generation</strong>, a <strong>countdown timer</strong>, <strong> audio </strong> for correct and incorrect answers, and <strong>background music</strong>.</p>

  <h2><u>Live Demo</u></h2>
  <p>You can view the live demo of the project-->   https://kubs-03.github.io/Flag_Whiz/</p>

  <h2>Features</h2>
  <ul>
    <li><strong>Randomized Questions:</strong> Generates a random set of questions each time the quiz starts using the Fisher-Yates shuffle algorithm.</li>
    <li><strong>Timer</strong>: A 15-second timer per question that can be toggled on or off.</li>
    <li><strong>Audio Feedback</strong>: Plays different sounds when the user selects a correct or incorrect answer.</li>
    <li><strong>Background Music</strong>:A looping background music that can be toggled on and off.</li>
    <li><strong>Responsive Design </strong>:The quiz is styled for both desktop and mobile devices.</li>
  </ul>

  <h2>How to Play</h2>
  <ol>
    <li>Click the Start Quiz button to begin.</li>
    <li>Choose the correct answer for each question before the timer runs out.</li>
    <li>The game will provide immediate feedback with sounds indicating if the selected answer is correct or incorrect.</li>
    <li>Move to the next question by clicking the Next button or waiting for the timer to expire.</li>
    <li>At the end of the quiz, your total score will be displayed.</li>
  </ol>

  <h2>Technologies Used</h2>
  <ul>
    <li><strong>HTML</strong>: Structure of the quiz game.</li>
    <li><strong>CSS</strong>: Styling for responsive design on desktop and mobile.</li>
    <li><strong>JavaScript</strong>: Game logic, random question generation, timer, and audio control.</li>
  </ul>

  <h2>How to Run the Project</h2>
  <ol>
    <li>Clone the repository:
      
      git clone https://github.com/Kubs-03/Flag_Whiz.git
  </li>
   <li>Open the index.html file in your browser to start the quiz.</li></ol>

  <h2>Customization</h2>
  <ul>
    <li><strong>Adding New Questions</strong>: You can add new questions in the questions array in the JavaScript file.</li>
    <li><strong>Changing Background Music</strong>: Replace the game_music.mp3 file in the audio folder with your own music file.</li>
    <li><strong>Updating Sound Effects</strong>: Modify the correct-sound and incorrect-sound audio files to change the feedback sounds.</li>
  </ul>
